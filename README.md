# Wine Classification using Naive Bayes

## Project Overview

This project demonstrates wine classification using the Naive Bayes algorithm. The Wine dataset is loaded from Scikit-Learn and used to train and evaluate classification models.

## Dataset

The dataset is provided by Scikit-Learn's built-in Wine dataset.

Features include:

* Alcohol
* Malic Acid
* Ash
* Alcalinity of Ash
* Magnesium
* Total Phenols
* Flavanoids
* Nonflavanoid Phenols
* Proanthocyanins
* Color Intensity
* Hue
* OD280/OD315
* Proline

Target classes represent different wine categories.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Jupyter Notebook

## Models Used

### Gaussian Naive Bayes

Used for continuous numerical features.

### Multinomial Naive Bayes

Applied for comparison purposes.

## Workflow

1. Load Wine dataset
2. Convert dataset into a Pandas DataFrame
3. Split data into training and testing sets
4. Train Gaussian Naive Bayes model
5. Evaluate model accuracy
6. Train Multinomial Naive Bayes model
7. Compare model performance

## Results

The Gaussian Naive Bayes model achieved high classification accuracy on the Wine dataset.

## Conclusion

Naive Bayes is a simple and effective machine learning algorithm for classification tasks. In this project, Gaussian Naive Bayes performed well because the dataset contains continuous numerical features.
